## Getting started
### Install and run locally via NPM
1. Clone the repository
2. Install dependencies with `npm install`
3. Run the development server with `npm run dev`

```bash
git clone https://github.com/TechOptimum/Tech-Optimum-Website
npm i
npm run dev
```
